# Chill Guy Clicker

Simple clicker game built with HTML, CSS, and JavaScript.  
The game allows you to earn points by clicking the Chill Guy, buy upgrades, and unlock an auto-clicker for passive income.

---

## 📌 About the Project

This project is a simple browser clicker game created to practice JavaScript game logic and DOM manipulation.

The game includes:
- Manual clicking system
- Upgrade system (click power)
- Auto-clicker for passive income
- Reset functionality
- Simple UI layout with Flexbox

The project helped me understand how to build interactive web applications using JavaScript.

---

## 🛠️ Technologies Used

- HTML5 – structure of the game
- CSS3 – layout, Flexbox, animations, styling
- JavaScript (ES6) – game logic, events, timers

---

## 🎮 Game Features

- Click Chill Guy to earn points
- Upgrade click power (+1 per upgrade)
- Buy auto-clicker (passive income per second)
- Reset game progress
- Simple and responsive UI layout

---

## 📁 Project Structure

- index.html – main game structure  
- style.css – styling and layout  
- script.js – game logic  
- assets/ – images (chill guy, icons, background)
